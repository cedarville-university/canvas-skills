# Skills

This repository ships Codex skills. Copy the skill(s) you want into your Codex
skills directory.

## Install

```bash
cp -R skills/canvas-tools ~/.codex/skills/
```

If your Codex home is different, replace `~/.codex` with your `CODEX_HOME`.

## Available Skills

- `canvas-tools`: Canvas LMS operations via `canvasapi`
